The Sea Wolf API

Overview and Quick Start Guide

Stay up to date with The Seawolf's latest products!

This is a web API that returns information about all the products that we offer. This API only accepts GET Requests for non-admin users.

With no inputs, it will return a list of all products in our database. For information on a specific product, use the product's id number as an input.

JSON

http://seawolf-shop-5-env-1.eba-xzwhb5mc.ap-southeast-2.elasticbeanstalk.com/api/?format=json

Django REST Framework

http://seawolf-shop-5-env-1.eba-xzwhb5mc.ap-southeast-2.elasticbeanstalk.com/api/?format=api

Example Code using Python


                import urllib2
                import json

                link = "http://seawolf-shop-5-env-1.eba-xzwhb5mc.ap-southeast-2.elasticbeanstalk.com/api/?format=json"

                req = urllib2.Request(link)
                response = urllib2.urlopen(req)

                obj = json.loads(response.read())

                print(len(obj))
                print(obj[0]["name"])
